#  Battleship Game (TypeScript Version)

## Overview

This is a browser-based Battleship game rebuilt using TypeScript. The original version was written in JavaScript, and this version was recreated to deepen understanding of TypeScript and improve code structure.

The project uses a modular architecture and focuses on applying object-oriented programming concepts together with strong typing.

---

## Features

* Interactive Battleship game (Player vs Computer)
* Drag & drop ship placement
* Random ship placement system
* Ship rotation (horizontal / vertical)
* Turn-based attack system
* Hit, miss, and sunk logic
* End-game detection with restart option
* Dynamic DOM rendering
* Clean separation between game logic and UI

---

## Technologies Used

* TypeScript
* JavaScript (ES6+)
* Vite
* HTML
* CSS

---

## Concepts Practiced

* Object-Oriented Programming (OOP) with classes
  (e.g. Ship, Gameboard, UI helpers)

* TypeScript features:

  * Interfaces
  * Type aliases
  * Strict typing
  * Generics (where applicable)

* DOM manipulation with type safety

* Event-driven architecture

* Modular file structure

* Separation of concerns (logic and UI)

## Purpose 
The purpose of this project was to rebuild an existing JavaScript project using TypeScript to improve code structure,
type safety, and overall development skills.
